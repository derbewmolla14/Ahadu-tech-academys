📚 Ahadu Tech Academy

Ahadu Tech Academy is a full-stack educational learning platform designed to deliver structured digital education from Elementary level up to University level.

The platform provides organized learning paths, video lessons, PPT materials, notes, and interactive course management for students in Ethiopia and beyond.

🎯 Project Features
👨‍🎓 User System
Student & Admin roles
Secure email-based authentication
Admin approval system before access
🏫 Education Structure
Elementary School (Grades 1–8)
High School (Grades 9–12)
University / College System
🎓 University Module
Ethiopian Universities list (e.g. AAU, Wollo, Bahir Dar, Jimma, etc.)
Searchable university selection system
Academic levels:
Natural Science
Social Science
Masters
PhD
🔬 Natural Science

Informatics (Computer Science, Software Engineering, IT, IS)

📖 Social Science

📺 Learning Content
Video lessons

🔒 Admin Features

💻 Coding Practice Feature

Express.js

🔐 Security

🎯 Purpose

This project is built to modernize education by providing a centralized digital learning system where students can access structured academic content from school to university level in one platform.
